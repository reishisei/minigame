# 塔防游戏扩展 - 配置实施指南

## 概述
本文档提供在 Cocos Creator 编辑器中配置新开发组件的完整指南。脚本开发已完成，现在需要在编辑器中配置组件属性和资源引用。

## 配置顺序建议
建议按以下顺序进行配置：

1. **GameManager** - 核心管理系统
2. **WaveManager** - 波次管理系统
3. **AudioManager** - 音频系统
4. **Slot组件** - 塔建造系统
5. **LevelManager** - 关卡系统
6. **SkillManager** - 技能系统
7. **AchievementManager** - 成就系统

## 配置前准备

### 1. 备份项目
在开始配置前，建议备份当前项目。

### 2. 检查脚本更新
确保所有脚本文件已更新到最新版本。

### 3. 资源准备
准备好需要的资源：
- 音频文件（.mp3/.wav）
- 敌人预制体（基于现有Enemy.prefab创建）
- UI元素（确保场景中有对应的节点）

## 配置步骤详解

**重要提示**：每个组件的UI配置都有详细说明。如果在配置UI引用时遇到困难，请参考对应组件的配置指南中的"UI配置详细步骤"部分。

### 第一步：配置 GameManager
1. 找到或创建 GameManager 节点
2. 添加 GameManager 脚本组件
3. 绑定 UI 引用（金币Label、经验Label等）**← 详细步骤见GameManager配置指南**
4. 验证基础功能

### 第二步：配置 WaveManager
1. 创建6种敌人预制体（basic, fast, armored, flying, boss, swarm）
2. 配置 WaveManager 节点
3. 绑定敌人预制体引用
4. 配置波次数据 **← WaveManager现在支持编辑器配置**
5. 测试敌人生成
6. （可选）配置UI引用 **← 详细步骤见WaveManager配置指南**

### 第三步：配置 AudioManager
1. 导入音频文件
2. 创建 AudioManager 节点
3. 添加4个 AudioSource 组件
4. 绑定音频剪辑
5. 测试音频播放

### 第四步：更新 Slot 组件
1. 找到所有 Slot 节点
2. 确保脚本已更新
3. 检查预制体引用（包含所有新塔类型）
4. 配置UI元素和按钮事件 **← 详细步骤见Slot配置指南**
5. 验证塔建造事件

### 第五步：配置 Shop 组件
1. 找到 Shop 节点
2. 确保脚本已更新
3. 配置UI引用和按钮事件 **← 详细步骤见Shop配置指南**
4. 验证商店功能

### 第六步：配置其他管理器
1. 创建 LevelManager、SkillManager、AchievementManager 节点
2. 添加对应脚本组件
3. 这些组件主要在代码中配置
4. （可选）配置技能和成就UI **← 详细步骤见SkillManager/AchievementManager配置指南**

## 常见问题解决

### 1. 属性引用丢失
**症状**：运行时控制台报错"xxx is null"
**解决**：在编辑器中重新绑定节点引用

### 2. 预制体引用错误
**症状**：生成敌人或塔时报错
**解决**：检查预制体路径和组件配置

### 3. 音频播放问题
**症状**：没有声音或声音异常
**解决**：检查音频文件格式、AudioSource配置

### 4. 事件不触发
**症状**：成就/技能系统没有反应
**解决**：检查GameManager事件调用，确保管理器已初始化
