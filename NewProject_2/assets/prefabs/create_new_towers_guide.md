# 新塔类预制体创建指南

以下四个新塔类已实现，需要在 Cocos Creator 编辑器中创建对应的预制体：

## 已创建的脚本类
1. **Mage** (法师) - 范围伤害塔
2. **Ice** (寒冰塔) - 减速效果塔  
3. **Support** (支援塔) - 增益光环塔
4. **Sniper** (狙击塔) - 长距离高伤害塔

## 创建预制体步骤

### 1. 创建空节点
- 在场景中创建一个空节点
- 命名为 "Mage", "Ice", "Support", "Sniper" 分别对应

### 2. 添加组件
- 为每个节点添加对应的脚本组件：
  - Mage 节点：添加 `Mage` 脚本
  - Ice 节点：添加 `Ice` 脚本  
  - Support 节点：添加 `Support` 脚本
  - Sniper 节点：添加 `Sniper` 脚本

### 3. 添加精灵(Sprite)组件
- 为每个节点添加 Sprite 组件
- 设置不同的颜色以区分类型：
  - Mage: 紫色 (128, 0, 255)
  - Ice: 浅蓝色 (0, 200, 255)
  - Support: 黄色 (255, 255, 0)
  - Sniper: 白色 (255, 255, 255)

### 4. 创建预制体
- 将每个节点拖拽到资源管理器中的 `assets/prefabs/` 文件夹
- 确保命名为：
  - Mage.prefab
  - Ice.prefab
  - Support.prefab
  - Sniper.prefab

### 5. 配置 Slot 组件
- 找到场景中的 Slot 节点（应该有多个）
- 在 Slot 组件的属性面板中，将新创建的预制体拖拽到对应的属性：
  - magePrefab: 拖拽 Mage.prefab
  - icePrefab: 拖拽 Ice.prefab
  - supportPrefab: 拖拽 Support.prefab  
  - sniperPrefab: 拖拽 Sniper.prefab

## 商店UI集成

### 1. 添加购买按钮
在 Shop 面板中需要添加4个新按钮：
- 购买法师按钮：调用 `onBuyMage()` 方法
- 购买寒冰塔按钮：调用 `onBuyIce()` 方法
- 购买支援塔按钮：调用 `onBuySupport()` 方法
- 购买狙击塔按钮：调用 `onBuySniper()` 方法

### 2. 按钮设置示例
每个按钮应设置：
- 显示文本：如 "法师 180G"
- 按钮事件：点击时调用对应方法
- 可根据塔类型设置不同的按钮颜色

## 测试建议
1. 确保所有预制体正确配置到 Slot 组件
2. 测试每种塔的购买功能
3. 验证特殊效果工作正常：
   - Mage: 攻击时对周围敌人造成范围伤害
   - Ice: 被攻击敌人移动速度降低
   - Support: 周围友军塔获得伤害和攻速加成
   - Sniper: 有几率造成暴击，可穿透多个敌人

## 注意事项
- Support 塔的光环效果需要其他塔在范围内才能生效
- Ice 塔的减速效果会在敌人移动速度恢复后消失
- Sniper 塔的穿透伤害会递减（每次减少30%）
- 所有塔的最终属性受 GameManager 全局加成影响
